# characters/

登場人物の設定資料を格納するディレクトリ。

## 運用ルール

- 1キャラ1ファイル: `<id>.md`（半角英数・小文字・ハイフン）
- ファイル先頭にキャラクター名・初登場話・関連キャラ・1行サマリーを置く
- 詳細はその後に自由形式で記述

## 推奨テンプレート

```markdown
# <表示名>

- ID: <id>
- 初登場: ep-NNN（タイトル）
- 関連: <他キャラID>
- 一行: <一文の要約>

## 概要

## 来歴

## 関係性

## 備考
```
