## ディレクトリ構成

```
noise-preservation-zone/
├── main/      # 確定稿（公開・連載用）
├── drafts/    # 推敲中の原稿（_v1, _v2 ...）
└── notes/     # 構想・断片・周辺メモ
```

## 運用ルール

- `main/` は確定稿。直接編集せず、`drafts/` で推敲してから差し替える。
- `drafts/` は `NN_タイトル_vN.md` 形式で版を重ねる。
- `notes/` は自由形式。構想・引用・断片を残す。
