# branches/

各分岐ルート（パラレル展開）の説明を格納するディレクトリ。

このディレクトリは**メタ情報**を扱う。実際のルート本文は git ブランチ `route/<name>` に置く。

## 運用ルール

- 1ルート1ファイル: `<route-name>.md`
- 分岐点・テーマ・現状ステータス・対応ブランチ名を明記
- 新規ルートを起こすときは、ブランチ作成と同時にここへ登録

## 推奨テンプレート

```markdown
# <ルート名>

- ブランチ: `route/<name>`
- 分岐点: 第NNN話「<タイトル>」直後
- 状態: drafting / paused / merged / abandoned
- 概要: <一段落>

## 動機

## 主要差分

## 現状の進捗
```
