# archive/

ボツ案・破棄稿・過去版を格納するディレクトリ。

`main/` から外したが完全に捨てたくない原稿、検討の途中で生まれた未採用案を残す場所。

## 運用ルール

- 命名: `NNN_<title>_<reason>.md`（例: `012_なんでもない一日_v1-discarded.md`）
- ファイル先頭に「経緯」「破棄理由」「採用版へのリンク」を記載
- 完全に消す前にここを経由させる（取り戻せる状態にしておく）

## 推奨テンプレート

```markdown
# <タイトル>（boneyard）

- 元位置: main/NNN_xxx.md
- 破棄日: YYYY-MM-DD
- 採用版: main/NNN_xxx.md（or route/<name>）
- 理由: <なぜ採用しなかったか>

---

<本文>
```
