# timeline/

時系列・年表を格納するディレクトリ。

## 運用ルール

- 本線時系列: `main-timeline.md`
- 分岐ルート時系列: `<route-name>-timeline.md`（例: `shenzhen-timeline.md`）
- 年表形式: `YYYY-MM-DD`（不明な場合は `YYYY-MM` や `YYYY`）+ 出来事 + 関連話

## 推奨テンプレート

```markdown
# <ライン名> 時系列

| 日付 | 出来事 | 関連話 |
|---|---|---|
| 2026-03 | チェンマイ到着 | ep-001 |
```
