# media/

映像化・メディア展開に関する設定資料を格納するディレクトリ。

## サブディレクトリ

- `anime/` — アニメ化想定資料（キャラデザ指針、画コンテ用要約、回想配分など）
- `movie/` — 実写映画想定資料（脚本骨子、ロケ提案、尺配分など）
- `visual/` — ビジュアル設定（コンセプトアート指針、配色、UI/モーション指示）
- `soundtrack/` — 音楽指針（BGM方向性、シーン別ムード、参考曲）

## 運用ルール

- 各サブディレクトリに `_index.md` を置き、用途を明記
- 提案版ごとにバージョンタグを切る: `pitch/anime-v1` など
- 実バイナリ（画像・音源）は原則含めず、外部ストレージのリンクで管理（リポジトリ肥大化防止）
